# **App Name**: Client Data Manager

## Core Features:

- Customer Data Input: Allow manual entry of customer data including card number, name, phone number, and customer type.
- Data Validation: Validate entered data to ensure correct formatting and completeness.
- SQL Server Connection: Establish a secure connection to the SQL Server database.
- SQL Query Execution: Execute SQL UPDATE statements to record client information into the database.
- Confirmation Display: Display a confirmation message to the user after successful data insertion.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey trust and reliability.
- Background color: Light Gray (#F5F5F5), creating a clean and professional look.  It is of similar hue as the primary color, but heavily desaturated, for a unified look.
- Accent color: Amber (#FFC107), used for important UI elements like confirmation buttons. Analogous to the primary, and different enough to create contrast.
- Font pairing: 'PT Sans' (sans-serif) for both headlines and body, ensuring readability and a modern aesthetic.
- Use flat, simple icons to represent customer data fields and actions, improving usability.
- A clean, single-column layout that prioritizes ease of data entry. Utilize clear labels and sufficient spacing.
- Subtle animations on data submission and confirmation to provide user feedback.
